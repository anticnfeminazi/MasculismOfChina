**女性主义**

> 女权主义（英语：feminism），又称女性主义，是指主要**以女性经验为来源与动机**，追求性别平权的社会理论与政治运动，包括消除性别定型观念，争取为妇女创造与男子平等的教育和职业**机会**等。在对社会关系进行批判之外，许多女权主义的支持者也着重于性别不平等的分析以及推动**妇女的权利、利益与议题**。



**男性主义**

> 男性主义（英语：Masculism），又作男权主义运动，是指一个主要**以男性经验为来源与动机**的社会理论与政治运动。男性主义并**不将女性视为敌人**，并且为在父权制下被歧视、压迫、忽略的男性受害者提供保护、支持、鼓励与协助。[1]认为男性不仅在父权制下遭受许多必须牺牲自然性的压抑与歧视，更反对一切形式的沙文主义。男性主义认为男性应获得聚焦，快速协助、保护、照顾、支援弱势男性。男性主义是与女性主义平行发展的。[2]由于派别不同，因此在对待女性主义上也不尽相同，例如乔治·贾尔德的作品就多为对女权运动的批判。对社会关系进行批判之外，许多男性主义的支持者也着重于性别不平等的分析以及推动男性的权利、利益与议题。在香港，香港男士协会是主要的男权运动组织。而起步较晚的台湾，除了台湾父亲权益协会、台湾男性协会，亦启发了英雄爸爸等新创公司以男性为主要受众[3]。

### 为什么男尊女卑已经过时

男尊女卑认为男性必定优于女性，随着时代的发展越来越缺乏合理性。之所以曾经认为男性优于女性，是因为男性整体体力强于女性，而历史上体力对于生存发展确实至关重要。世易时移，一方面人类种群的存在已经不再是一个问题，更有条件越来越关注每个个体，而性别并非体力的决定性因素；另一方面，大机器生产代替手工劳动，体力的作用不断降低。认为男性优于女性越来越站不住脚，逐渐受到挑战。

### 中国网络女权主义有哪些类别

“自由女权主义”：认为女性本质上和男性一样优，乃至认为女性已经本质上优于男性。常常主张男女不平等是历史上和当下男性阴谋的结果。认为男女不平等是男性有意压迫的结果。反抗男权压迫实现机会平等。（如：认为男性能做的女性也能做，主张取消个别领域对女性的比例限制）

”社会女权主义“：关注实现平等，而非仅仅把女性和男性一样对待。认为应该关注和满足女性的特别需求，采取积极措施实现结果平等。（如：卫生巾免税乃至免费，性别薪酬差距，高管女性比例）

“激进女权主义”：认为父权是造成社会最严重问题的根本原因，乃至认为男性存在本身是造成社会问题的根本原因（男性原罪论），甚至进一步的仇视个体男性，认为个体男性对长期以来的男女不平等负有责任，即使并不是积极加害者。批评女性传统性别角色的同时，同样批评扮演传统性别角色的个体女性。（如：女权男向女性道歉，反婚反育，反对做家庭妇女）

### 什么是语言腐败

**语言腐败 百度百科**

> 所谓语言腐败，指一些拥有话语权的人为了相关利益和意识形态的目的，偷换语言的概念，将一些词汇的含义做一些完全相反的解释。
>
> 所谓语言腐败，是指具有话语权的人出于经济的、政治的、意识形态的目的，随意改变词汇的含义，甚至赋予它们与原来的意思完全不同的含义，操纵人心。语言腐败的典型形式是冠恶行以美名，或冠善行以恶名。

比如认为女权主义是男女平等，或是维护女性正当权益。

但是认为男权主义是男尊女卑，不是维护男性正当权益。

并不认为男权主义是男女平等，或是维护男性正当权益。

破除语言腐败，应该从词语的字面本来意思出发，“女权”就是“女”的“权”，也就是女性的权利/权力，明显不是男女平等的意思。

### 为什么女权不是平权

女权是女性视角下的男女平等，女性视角更多关心对女性不利的一面和只对男性有利的一面，与完整的男女平等相比有明显的缺失。

历史上女性完全依附于男性，女性权力缺失的问题比对男性刻板印象的问题更严重。当下多数女性早已不再完全依附男性，法律上也不再允许男性要求女性完全依附，女性有和男性一样的基本权利。男性和女性同样受社会文化过时的刻板印象束缚，而非女性受制度束缚。因此女权不是平权。

有些自称的女权主义者奉行女性利己主义，明显不是女权主义，更不是平权主义。
